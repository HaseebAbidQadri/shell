# Python Mini Shell

## Overview
This project is a lightweight Python-based mini shell that replicates basic terminal functionalities. It allows users to execute commands, navigate files, and manage processes within a controlled environment. Designed for learning and experimentation with shell behavior.

## Features
- Execute standard system commands
- File and directory navigation
- Process management and control
- Simple and easy-to-understand Python code

## Tech Stack
- Python 3.11
- Standard Python libraries (no external dependencies)
